Fork y Pull Requests

Este documento busca aplicar las acciones fork y pull requests de Git y GitHub para agregar el nombre del estudiante a la lista

Para hacerlo deberás primero hacer un fork, para tener una versión integra del proyecto en tu espacio de usuario en GitHub

Luego deberás clonar el repo de tu espacio de usuario, a tu equipo usando git clone <url-repo>

Una vez en tu equipo deberás agregar tu nombre a la lista de usuarios, y hacer un commit con la modificación

Deberás sincronizar el repo local, con tu repo "forkeado", sincronizando ambos repositorios

Una vez sincronizados los repositorios, deberás enviar una solicitud pull request

¿Qué pasa después?

Si el dueño "orginal" del repositorio acepta los cambios, estos se verán reflejados en el repositorio desde el cual hiciste el fork
